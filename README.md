 End-to-End Data Engineering Pipeline

Transforming raw data into actionable insights through robust ETL processes and Star Schema modeling.

 🚀 Executive Summary

Engineered a robust, end-to-end data pipeline that automates the extraction, transformation, and loading of heterogeneous data sources—including XML, XLSX, CSV, and TXT—into a centralized PostgreSQL Data Warehouse. By leveraging Talend for ETL processing and applying a structured Star Schema, I ensured high data integrity and optimized performance for complex analytical queries. The project culminates in an intuitive BI dashboard that transforms these integrated datasets into actionable business insights, effectively streamlining the decision-making process.

 🛠 Tech Stack

* **ETL & Orchestration**: Talend Open Studio
* **Database**: PostgreSQL (Data Warehouse)
* **Modeling**: Star Schema (Fact & Dimension tables)
* **Visualization**: Tableau
* **Version Control**: Git & GitHub
